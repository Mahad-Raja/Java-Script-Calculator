# Basic Calculator

A simple and responsive calculator built using **HTML**, **CSS**, and **JavaScript**. This project performs basic arithmetic operations through an easy-to-use interface.

## 🚀 Features

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Clear (C) button
* Responsive and user-friendly design
* Real-time display updates

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```text
calculator/
│── index.html
│── style.css
│── script.js
└── README.md
```

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any modern web browser.

No installation or additional software is required.
